# Hitchhiker's Guide

A physical book containing a LilyGo T-Deck, creating a portable, tactile interface for digital inquiry. The Hitchhiker's Guide is both artifact and tool—a book you can type into.

## Project Page

This project has its own GitHub Pages site with a Matrix chat widget for talking with the project faculty lead.

Visit: https://InquiryInstitute.github.io/atelier-hitchhikers-guide/

## Chat Widget

The project page includes a Matrix chat widget that:
- Connects to Matrix for real-time chat
- Uses RAG (Retrieval Augmented Generation) to answer questions about the project
- Allows direct conversation with the project faculty lead
